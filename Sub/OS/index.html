DOS (Disk Operating System)

Disk Operating System (DOS) adalah keluarga sistem operasi berbasis teks yang mendominasi pasar komputer pribadi (PC) dari awal 1980-an hingga pertengahan 1990-an. Pengguna berinteraksi melalu antarmuka baris perintah (command-line), mengetikkan perintah untuk menjalankan program atau mengelola file. MS-DOS dari Microsoft adalah varian yang paling terkenal dan menjadi dasar bagi versi awal Windows.

Perkembangan Kunci:

  • 1980: QDOS (Quick and Dirty Operating System) diciptakan oleh Tim Paterson dari Seattle Computer Products.
  • 1981: Microsoft membeli QDOS, mengubah namanya menjadi MS-DOS, dan melisensikannya kepada IBM untuk PC pertama mereka (dirilis sebagai PC-DOS).
  • 1983: MS-DOS 2.0 dirilis, menambahkan dukungan untuk hard drive dan subdirektori, sebuah lompatan besar dari sistem berbasis disket.
  • 1995: Popularitas DOS menurun drastis setelah peluncuran Windows 95, yang merupakan sistem operasi grafis mandiri.

Fakta Unik: Bill Gates awalnya menyarankan IBM untuk menggunakan sistem operasi CP/M yang populer saat itu. Namun, karena negosiasi gagal, Microsoft mengambil kesempatan untuk menyediakan OS mereka sendiri.

Microsoft Windows

Microsoft Windows adalah sistem operasi antarmuka grafis (GUI) yang paling dominan di dunia untuk komputer pribadi. Awalnya dirilis sebagai "lapisan" di atas MS-DOS, Windows berevolusi menjadi sistem operasi mandiri yang kuat, memperkenalkan konsep seperti desktop, ikon, dan Start Menu kepada jutaan pengguna.

Perkembangan Kunci:

  • 1985: Windows 1.0 dirilis sebagai lingkungan operasi grafis pertama dari Microsoft.
  • 1995: Windows 95 menjadi sebuah fenomena global, memperkenalkan Start Menu, taskbar, dan arsitektur 32-bit.
  • 2001: Windows XP dirilis, menggabungkan stabilitas dari kernel NT dengan antarmuka yang ramah pengguna. Menjadi salah satu versi Windows paling dicintai.
  • 2015: Windows 10 diperkenalkan dengan model "Windows as a Service", menyatukan pengalaman di berbagai perangkat dan menerima pembaruan berkelanjutan.

Fakta Unik: Nama kode internal untuk Windows 95 adalah "Chicago".

Apple macOS

macOS (sebelumnya Mac OS X dan OS X) adalah sistem operasi yang dikembangkan oleh Apple Inc. secara eksklusif untuk jajaran komputernya. Dikenal dengan desain antarmuka yang elegan, kemudahan penggunaan, dan fondasi berbasis UNIX yang kuat, macOS menjadi pilihan utama bagi banyak profesional kreatif.

Perkembangan Kunci:

  • 1984: "System Software" debut bersama komputer Macintosh pertama, mempopulerkan antarmuka grafis (GUI) secara massal.
  • 2001: Mac OS X 1.0 "Cheetah" dirilis, menandai transisi ke arsitektur berbasis UNIX (dari NeXTSTEP) yang jauh lebih stabil dan modern.
  • 2007: Integrasi dengan iOS dimulai, membawa banyak fitur dari iPhone ke Mac.
  • 2020: Apple memulai transisi dari prosesor Intel ke chip buatan sendiri (Apple Silicon), dan macOS Big Sur dirilis dengan desain ulang terbesar dalam hampir 20 tahun.

Fakta Unik: Sebelum dinamai berdasarkan lokasi di California (seperti Mavericks, Yosemite), semua versi Mac OS X (10.0-10.8) dinamai berdasarkan spesies kucing besar.

Linux

Linux adalah keluarga sistem operasi mirip Unix yang bersifat open-source dan dikembangkan secara kolaboratif. Awalnya hanya sebuah kernel yang dibuat oleh Linus Torvalds, Linux kini menjadi tulang punggung dari sebagian besar infrastruktur internet, superkomputer, dan perangkat IoT. Ia dikenal karena stabilitas, keamanan, dan fleksibilitasnya.

Perkembangan Kunci:

  • 1991: Linus Torvalds mengumumkan proyek hobinya untuk membuat kernel OS baru di Usenet.
  • 1992: Kernel Linux digabungkan dengan komponen dari Proyek GNU, menciptakan sistem operasi GNU/Linux yang lengkap dan fungsional.
  • 1993: Distribusi (distro) besar pertama seperti Slackware dan kemudian Debian muncul, memudahkan instalasi dan penggunaan Linux.
  • 2004: Canonical merilis Ubuntu, distro yang berfokus pada kemudahan penggunaan dan mempopulerkan Linux di kalangan pengguna desktop.

Fakta Unik: Hampir semua superkomputer tercepat di dunia (TOP500) menjalankan varian dari sistem operasi Linux.

Android

Android adalah sistem operasi seluler berbasis kernel Linux yang dimodifikasi. Dikembangkan oleh Google, Android bersifat open-source, memungkinkan ribuan produsen perangkat untuk menggunakannya. Hal ini menjadikannya OS seluler paling populer di dunia, digunakan di smartphone, tablet, TV, dan banyak lagi.

Perkembangan Kunci:

  • 2005: Google mengakuisisi perusahaan startup bernama Android Inc.
  • 2008: HTC Dream (T-Mobile G1) dirilis sebagai ponsel Android komersial pertama.
  • 2010: Android 2.2 "Froyo" dirilis, membawa peningkatan kecepatan signifikan dan dukungan untuk fitur seperti hotspot Wi-Fi.
  • 2014: Android 5.0 "Lollipop" memperkenalkan bahasa desain "Material Design" yang menjadi ciri khas antarmuka Android modern.

Fakta Unik: Setiap versi mayor Android (dari 1.5 hingga 9) dinamai berdasarkan makanan manis atau pencuci mulut secara alfabetis (Cupcake, Donut, Eclair, Froyo, ...).
