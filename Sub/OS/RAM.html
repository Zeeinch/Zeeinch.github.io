Apa itu RAM? RAM (Random Access Memory) adalah memori utama komputer yang bersifat "sementara" (volatile). Fungsinya adalah menyimpan data dan instruksi program yang sedang aktif digunakan oleh CPU. Data ini hilang saat komputer dimatikan. [Lompat ke video]

Bagaimana RAM Bekerja (Alur Data) Saat Anda membuka program, data program tersebut disalin dari penyimpanan permanen (seperti Hard Drive atau SSD) ke RAM. CPU kemudian mengakses data super cepat dari RAM untuk bekerja. [Lompat ke video]

Mengapa RAM Besar Membuat Komputer Cepat? Jika RAM penuh, komputer terpaksa menggunakan Hard Drive (yang jauh lebih lambat) sebagai "memori" cadangan. Proses bolak-balik ke Hard Drive inilah yang membuat komputer terasa lambat. RAM yang lebih besar menyediakan ruang kerja yang lebih luas sehingga CPU tidak perlu "menunggu" data dari Hard Drive. [Lompat ke video]

Modul RAM: DIMM Secara fisik, RAM dipasang di motherboard menggunakan modul yang disebut DIMM (Dual Inline Memory Module). Modul ini memiliki pin di kedua sisinya. Jumlah pin bervariasi tergantung generasinya (misal, DDR3 memiliki 240 pin, DDR4 memiliki 288 pin). [Lompat ke video]

Jenis RAM: DRAM vs SDRAM

  • DRAM (Dynamic RAM): Jenis RAM dasar yang menyimpan data di kapasitor. Kapasitor ini "bocor" sehingga harus di-refresh secara konstan (dinamis) agar datanya tidak hilang. [Video]
  • SDRAM (Synchronous DRAM): Versi DRAM yang lebih modern dan lebih cepat karena kecepatannya disinkronkan dengan jam (clock) sistem komputer. [Video]

Pentingnya Jalur Data: 64-bit (DIMM) vs 32-bit (SIMM) Modul DIMM modern memiliki jalur data 64-bit, artinya mereka dapat mentransfer 64 bit data sekaligus dalam satu siklus jam. Ini dua kali lebih cepat dari modul SIMM (Single Inline Memory Module) lama yang hanya 32-bit. [Lompat ke video]

Memahami Kecepatan & Bandwidth (Contoh: PC100) Kecepatan RAM (misal, 100 MHz) dikalikan dengan lebar bus (8 byte untuk 64-bit) menghasilkan bandwidth. Contoh: RAM PC100 (100 MHz) x 8 byte = 800 Megabyte per detik (MB/s). [Lompat ke video]

Revolusi DDR (Double Data Rate) Teknologi DDR secara efektif menggandakan kecepatan transfer data. Jika RAM biasa (SDR) hanya mentransfer data pada "sisi naik" (rising edge) sinyal jam, DDR mentransfer data di sisi naik *dan* sisi turun (falling edge), sehingga menggandakan bandwidth tanpa harus menggandakan frekuensi jam. [Lompat ke video]

Generasi DDR: DDR2, DDR3, DDR4 Setiap generasi baru (DDR2, DDR3, DDR4) menawarkan kecepatan yang lebih tinggi, bandwidth lebih besar, dan konsumsi daya yang lebih rendah. Meskipun DDR2 dan DDR3 sama-sama memiliki 240 pin, posisi "takik" (notch) mereka berbeda sehingga tidak dapat ditukar. DDR4 memiliki 288 pin. [Lompat ke video]

Memori Khusus: ECC RAM ECC (Error Correcting Code) RAM adalah jenis memori khusus yang dapat mendeteksi dan memperbaiki kerusakan data kecil secara otomatis. Memori ini sangat penting untuk server dan sistem kritis (keuangan, medis) di mana korupsi data tidak dapat ditoleransi. Modul ECC dapat diidentifikasi secara fisik karena biasanya memiliki 9 chip memori (8 untuk data, 1 untuk ECC), bukan 8 chip seperti RAM non-ECC. [Lompat ke video]
